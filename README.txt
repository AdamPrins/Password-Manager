INSTALLATION
==========================
 - Ensure all pip modules are installed by navigating into the src folder
   and running the following command
       WINDOWS
       pip install -r requirements.txt

       LINUX/MAC
       pip3 install -r requirements.txt

RUNNING
==========================
- Navigate into the src folder and run:
       WINDOWS
       python ./main.py

       LINUX/MAC
       python3 ./main.py
